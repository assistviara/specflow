# SpecFlow Vision

## Vision

SpecFlowは、

AIによるソフトウェア開発をオーケストレーションするための
開発基盤である。

SpecFlowは、
仕様・設計・実装・テスト・レビューを
一つの開発プロセスとして統合し、

人間の意思決定と
AIの実装能力を組み合わせることで、

変更に強く、
品質を継続的に改善できる
ソフトウェア開発を実現する。

SpecFlow自身も、
SpecFlowのプロセスに従って開発される。

## Principles

- 人間が要求と意思決定を行う
- AIが実装・テスト・レビューを支援する
- 仕様・設計・実装・レビューを資産として残す
- 小さく実装し、小さく検証する
- 品質はテストとレビューの両方で保証する
- SpecFlow自身もSpecFlowで開発する
