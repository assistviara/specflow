# Prompt Templates

このフォルダには、  
AIへ渡すPromptの共通テンプレートを格納する。

Promptはコードではなく、  
SpecFlowの重要な資産である。

各プロジェクトでは、  
Templateを基にPromptを生成する。

## 役割

- Templateは共通ルールを定義する
- Promptはプロジェクト固有の内容を持つ
- PromptはGitで管理し、変更履歴を残す
- Promptの変更もレビュー対象とする

## 格納予定ファイル

- `plan_prompt_template.md`
- `implement_prompt_template.md`
- `review_prompt_template.md`
- `repair_prompt_template.md`
